# CAR
A Rust library for reading, writing and manipulating [IPLD Content Archive files](https://ipld.io/specs/transport/car/)

## Feature support
- [x] Read CAR v1
- [x] Read CAR v2
- [ ] Write CAR v1
- [ ] Write CAR v2
- [ ] Split CAR files i.e. [carbites](https://github.com/nftstorage/carbites)
